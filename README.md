# Learnova
Learnova is an AI powered learning assistant designed to make study faster, smarter and more interactive.also we can store study time in it for our improvement.
It is mostly for the stuents who have important exams like Boards, End term examinations. It contains only 4 subjects till now that are Physics, Chemistry, Mathematics, Biology. In future we can add more subjects like Computer science, Physical education, Hindi, etc.
